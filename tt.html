Hexo Next主题TOC功能示例

展示文章目录功能配置与效果

简介

Hexo是一个快速、简洁且高效的博客框架，而Next则是Hexo中最受欢迎的主题之一。它提供了丰富的功能和配置选项，让用户能够自定义博客的外观和功能。

TOC配置

通过简单的配置，Next主题可以自动为文章生成目录。以下是一个配置示例：

# Next主题TOC配置示例
toc:
  enable: true      # 启用目录
  number: true     # 显示编号
  wrap: false      # 自动换行
  expand_all: false # 展开所有级别
  max_depth: 6      # 最大深度

配置选项说明

enable: 设置为true以启用目录功能。

number: 设置为true会在目录项前显示编号。

wrap: 控制长标题是否自动换行。

expand_all: 设置为true会默认展开所有级别的标题。

max_depth: 决定目录中包含的标题级别深度，最大为6级。

使用方法

在文章Front-matter中，可以通过设置toc属性来控制是否显示目录：

---
title: 文章标题
toc: true
---

Front-matter选项

除了全局配置外，还可以在每篇文章的Front-matter中覆盖全局设置：

---
title: 文章标题
toc: true
toc_number: true
toc_max_depth: 4
---

自定义样式

如果需要修改目录的样式，可以通过自定义CSS来实现：

/* 自定义TOC样式 */
.post-toc {
  font-size: 14px;
  line-height: 1.8;
}

.post-toc .toc-item {
  margin: 5px 0;
}

.post-toc .toc-link {
  color: #6a11cb;
  text-decoration: none;
}

位置调整

默认情况下，TOC会显示在文章的右侧。在移动设备上，它会自动调整到文章上方。

移动端视图

Next主题已经为移动设备做了优化，TOC会以更紧凑的形式显示。

结论

Hexo的Next主题提供的TOC功能非常实用，可以帮助读者快速导航长篇文章。通过简单的配置，可以满足大多数博客的需求。
