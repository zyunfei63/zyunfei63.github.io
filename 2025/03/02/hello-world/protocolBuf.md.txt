#  安装

##  win10安装1

```txt
01 前提安装了Chocolatey 
02 choco install protoc
03 配置环境系统变量  具体位置，日志会打印
04 protoc-gen-go.exe 的安装 （用于将file.proto文件转译为file.pb.go文件）
         go install google.golang.org/protobuf/cmd/protoc-gen-go@latest
    

```



##   win10安装方式2

```
01 https://github.com/protocolbuffers/protobuf/releases/tag/v26.1
02 解压
03 与环境变量
04 protoc-gen-go.exe 的安装 （用于将file.proto文件转译为file.pb.go文件）
```
